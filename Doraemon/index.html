ชินบิ หอพักอลเวง โกสต์บอล Z ครึ่งแรก

7.3
12+ 24min

เรื่องผีของฮยุนวู SMS โด่งดังไปทั่วประเทศ คำขอเริ่มมีเข้ามาจากทั่วทุกมุม ฮาริและเพื่อนๆของเขาก็แก้ปัญหาด้วยการไล่ผีด้วยกัน แล้ววันหนึ่ง ชายลึกลับก็ปรากฏตัวขึ้นรอบๆ ฮาริและกลุ่มของเขา และความลับที่ถูกซ่อนไว้มานานก็เริ่มถูกเปิดเผยอย่างช้าๆ

Starring : ซอก จงซอ,วิศวกร บยองพิล,ชิน ดงจิน,ฮยอน โบนา,ชิน กิลจู,มา ซัง-ฮยอน
Genres : สยองขวัญ

ชินบิ หอพักอลเวง แวมไพร์แห่งแสงและบุตรแห่งความมืด

7.4
12+ 57min

ในวันที่สำนักงานใหญ่ขององค์กรขับไล่ผี อีจิสถูกโจมตีโดยแวมไพร์ 'ริออน' ได้พบกับบุคคลที่คุ้นเคย เขายังฝ่าฝืนคำสั่งของกัปตันที่ให้อยู่ห่างจากเหตุการณ์และไล่ตามเขาไป... ในขณะเดียวกัน 'คังริม', 'เอียน' และ 'ชินบี' ตัดสินใจผนึกกำลังเพื่อหยุดยั้งแผนการของแวมไพร์ชั่วร้าย 'บาร์โลว์' แต่ด้วยเหตุผลบางอย่าง ลีออนจึงขัดขวางเส้นทางของพวกเขา และความจริงก็ถูกเปิดเผย เพื่อปกป้องสิ่งล้ำค่า ลีอนต้องเผชิญหน้ากับตัวตนในอดีตและปลุกพลังที่แท้จริงของเขา!

Starring : ซอกจงซอ,กงบยองพิล,ลีโบฮยอน,ซนมินยอง,คิมมินอา
Genres : ผจญภัย
Tags : สยองขวัญ, ดราม่า, ผจญภัย

ชินบิ หอพักอลเวง เดอะมูฟวี่ โทเกบีเวลา ปะทะ ยอร์มุนกันต์

7.5
16+ 1h 30min

สองพี่น้อง 'ฮาริ' และ 'ดูริ' ไปเที่ยวกับครอบครัวช่วงวันหยุดฤดูหนาว เขาพบก็อบลินท้องฟ้า 'จูบิ' นอนอยู่บนปีกเครื่องบิน จึงเรียกชินบิและกึมบิ และแล้วการโจมตีที่ไม่รู้จักก็เริ่มต้นขึ้น สัตว์ประหลาดในตำนาน 'ยอร์มุนกันต์' ที่ฟื้นคืนชีพจากโลงศพในทะเลลึก พยายามที่จะกลืนกินเครื่องบินลำนั้น ในขณะเดียวกัน คังลิมผู้ได้รับเลือกให้เป็นผู้ไล่ผีบนท้องฟ้าก็ร่วมช่วยเหลือเพื่อนๆ ของเขา ผู้ต้องการทำลายโลก โจมตีอาณาจักรก็อบลินฟ้าอย่างรุนแรง 'ฮาริ' และผองเพื่อน' การผจญภัยบนท้องฟ้าอย่างหลีกเลี่ยงไม่ได้เริ่มต้นขึ้นแล้ว...

Genres : ผจญภัย
